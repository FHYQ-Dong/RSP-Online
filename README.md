# RSP-Online

## Repository

[Tsinghua Git](https://git.tsinghua.edu.cn/donghy23/extra-experiment-rsp-online)

[Github](https://github.com/FHYQ-Dong/RSP-Online)

## Usage

1. run server by

   ```bash
   server.exe <ip> <port> <account_file>
   ```
2. run client by

   ```bash
   client.exe
   ```
3. play under instruction!

## Build

### Requirements:

- cmake >= 3.12
- MinGW or MSVC compiler
